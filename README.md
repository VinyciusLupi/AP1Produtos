quando você inicia o software vai mostrar uma lista de produtos, em cada produto vai ter seu nome , preço e um botão escrtito favoritar ao clicar no botão e esse item
é salvo em uma lista que só tem produtos que você favoritou, para acessar essa lista você clica num botão no canto superior direito que diz favoritos, e para voltar
você clica no botão ao lado que diz home. Na pagina home antes de chegar nos produtos yem uma barra para pesquisa onde filtra os produtos por nome, voce escreve um
nome e só aparece produtos com aquele nome.
